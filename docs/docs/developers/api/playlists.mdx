## Get Trending Playlists

<Tabs
  block={true}
  defaultValue="request"

>

```bash
GET https://audius-discovery-3.theblueprint.xyz/v1/playlists/trending?app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param    | Type     | Required | Note                                           | Available values                |
| -------- | -------- | -------- | ---------------------------------------------- | ------------------------------- |
| time     | _string_ | false    | Calculate trending over a specified time range | `week` `month` `year` `allTime` |
