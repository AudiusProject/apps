## Get Track

Gets a track by ID

<Tabs
  block={true}
  defaultValue="request"
  values={[{"label":"Request","value":"request"},{"label":"Response","value":"response"}]}
>
<TabItem value="request">

```bash
GET https://audius-discovery-3.theblueprint.xyz/v1/tracks/D7KyD?app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param    | Type     | Required | Note          |
| -------- | -------- | -------- | ------------- |
| track_id | _string_ | true     | A track ID    |
| app_name | _string_ | false    | Your App Name |

</TabItem>
