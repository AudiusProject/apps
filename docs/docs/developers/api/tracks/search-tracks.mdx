<TabItem value="request">

```bash
GET https://audius-discovery-3.theblueprint.xyz/v1/tracks/search?query=baauer%20b2b&app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param             | Type     | Required | Note                            |
| ----------------- | -------- | -------- | ------------------------------- |
| query             | _string_ | true     | The search query                |
| only_downloadable | _string_ | false    | Return only downloadable tracks |
| app_name          | _string_ | false    | Your App Name                   |
