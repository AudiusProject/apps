## Get User Connected Wallets

Get the User's ERC and SPL connected wallets

<Tabs
  block={true}
  defaultValue="request"
  values={[{"label":"Request","value":"request"},{"label":"Response","value":"response"}]}
>
<TabItem value="request">

```bash
GET https://audius-discovery-6.cultur3stake.com/v1/users/Wem1e/connected_wallets?app_name=EXAMPLEAPP HTTP/1.1
```

> Path Parameters

| Param    | Type     | Required | Note          |
| -------- | -------- | -------- | ------------- |
| id       | _string_ | true     | A User ID     |
| app_name | _string_ | false    | Your App Name |

</TabItem>
<TabItem value="response">

`200: OK`

```json
{
  "data": {
    "erc_wallets": ["0x42b53313f643D2a8007a169C2eC973100A1F84C3"],
    "spl_wallets": ["EF1zneAqA2mwjkD3Lj7sQnMhR2uorGqEHXNtAWfGdCu2"]
  }
}
```

</TabItem>
</Tabs>
