## Proper Nouns

> The following are to be used in the upper case when refering to a user or entity.

- Good Example: "Audius is built to empower Artsts to distribute music to Fans directly."
- Bad Example: "Audius is built to empower artsts to distribute music to fans directly."

- Artist(s)
- Fan(s)
- Developer(s)

## Terminology

- `onchain` - no space, no hyphen
- `offchain` - no space, no hyphen
- `$AUDIO` - when refering to the Audius Network Token, always include `$` prefix and capitalize all
  letters
