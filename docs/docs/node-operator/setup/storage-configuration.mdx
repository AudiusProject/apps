## Storage Configurations

:::info Content Nodes Only

Storage configuration settings are only applicable to
[Content Nodes](/learn/architecture/content-node).

:::

The configuration file will be pre-populated with the required fields. Enter your information into
each field for each Audius Content Node you will be running.

<TabItem value="aws">

```bash showLineNumbers title="audius-ctl configuration file with AWS storage config"
network:
  deployOn: mainnet
nodes:
  content-1.example.com:        # <--- THE URL OF YOUR CONTENT NODE
    type: content
    privateKey: abc123          # <--- UNIQUE PRIV KEY USED BY THIS NODE TO SIGN RESPONSES
    wallet: 0xABC123            # <--- UNIQUE WALLET ADDRESS OF ABOVE PRIV KEY
    rewardsWallet: 0xABC123     # <--- ADDRESS OF WALLET HOLDING STAKED TOKENS
    storage:
      storageUrl:               # <--- REQUIRED
      awsAccessKeyId:
      awsSecretAccessKey:
      awsRegion:
```

</TabItem>
