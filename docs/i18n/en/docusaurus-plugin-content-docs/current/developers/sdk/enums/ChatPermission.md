Defines who the user allows to message them

## Enumeration Members

### ALL

 **ALL**

Messages are allowed for everyone

___

### FOLLOWEES

 **FOLLOWEES**

Messages are only allowed for users I follow

___

### NONE

 **NONE**

Messages are not allowed

___

### TIPPERS

 **TIPPERS**

Messages are only allowed for users that have tipped me
