## Use cases and examples

- Use Flex as a container to align and lay out elements without css.
- Use Box to wrap and size individual elements, then wrap in Flex to align a set of Box siblings.
- Use Flex around a single element to center or align it.
- Use Flex to add alignment spacing, and padding/margin props all at once. It accepts all the props from Box as well.
- Prefer Box if the flex props are not being used.
