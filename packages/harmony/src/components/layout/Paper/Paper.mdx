## Use cases and examples

- The paper component has pre-set properties for color, corner radius, shadow, border color, and spacing.
- Use the paper component to quickly create containers and eliminate guesswork.
- Ensure consistency by using the paper component to create containers that follow Harmony’s foundational properties.
