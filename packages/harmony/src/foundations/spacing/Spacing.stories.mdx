## Overview

Consistent and scalable spacing helps you eliminate guesswork because you're designing with a limited set of options.

## Spacing sizes

Using the right sizes will create consistency and hierarchy in user interface.

export function SpacingExample(props) {
  const { spacing } = props
  const theme = useTheme()
  const spacingValue = theme.spacing[spacing]
  return (
    <Flex gap='xl' direction='column' alignItems='center'>
      <Flex
        alignItems='center'
        justifyContent='center'
        h={spacingValue}
        w={64}
        css={{ background: 'rgba(255, 0, 0, 0.10)' }}
      >
        <Text css={{ color: '#F00', fontSize: Math.min(spacingValue, 10) }}>
          {spacingValue}
        </Text>
      </Flex>
      <Text>{spacing}</Text>
    </Flex>
  )
}

## Usage and examples

- Stay consistent with paddings and sizes across different UI components.
- Align all the elements to the same spacing within the components or a page.
- Elements should be grouped by likeness, with the most similar elements closer together and unlike elements farther apart.
