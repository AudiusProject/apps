<Heading> 
## Text Styles

Text styles are a pre-defined set of styles to apply to your designs: Display, Heading, Title, Label, and Body

</Heading>

| Name    | Description                                                      | Weight              | Sizes           | Color                                      |
| ------- | ---------------------------------------------------------------- | ------------------- | --------------- | ------------------------------------------ |
| Display | Display styles should be reserved for visual type elements       | Regular, Bold       | XL, L, M, S     | Heading, Default, Subdued, Disabled        |
| Heading | Used as the primary text on a page to indicate what the page is. | Regular, Bold       | XL, L, M, S     | Heading, Default, Subdued, Disabled        |
| Title   | Used as the secondary text on a page.                            | Thin, Regular, Bold | XL, L, M, S, XS | Heading, Default, Subdued, Disabled        |
| Label   | Used as an all caps label for elements.                          | Regular, Bold       | XL, L, M, S, XS | Heading, Default, Subdued, Disabled        |
| Body    | Used for body or descriptive text.                               | Regular, Bold       | L, M, S, XS     | Heading, Default, Subdued, Disabled, Error |

## Weight Mappings

Avenir Next LT Pro's font weights in Figma do not match perfectly to code. See an accurate mapping below and always double check the developer panel.

</Heading>

export function Emp(props) {
  return <Text {...props} color='danger' strength='strong' />
}

| Figma                 | Code                       |
| --------------------- | -------------------------- |
| Heavy - 900           | Heavy - 900                |
| <Emp>Bold - 800</Emp> | <Emp>Bold - 700</Emp>      |
| <Emp>Demi - 700</Emp> | <Emp>Demi Bold - 600</Emp> |
| Medium - 500          | Medium - 500               |
| Regular - 400         | Regular - 400              |
| Light - 300           | Light - 300                |
| Thin - 275            | Thin - 200                 |
| Ultra Light - 250     | Ultra Light - 100          |
